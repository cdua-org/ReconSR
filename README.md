# ReconSR

ReconSR is an automated OSINT tool currently in the initial design and development phase. The project aims to provide cybersecurity researchers with a modular framework for analyzing the attack surface of web applications and associated services.

This repository is under development. 

### Roadmap

- 🏗️ WHOIS data retrieval and analysis.
- 🏗️ DNS record retrieval and analysis.
- 🏗️ Mail security policy verification.
- SSL/TLS certificate investigation.
- Integration of external service APIs.
